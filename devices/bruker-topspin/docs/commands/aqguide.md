# aqguide

**Category:** Commands > Acquire > More

## NAME

**aqguide** - Open the Acquisition Guide


## DESCRIPTION

The command aqguide opens the TopSpin Acquisition Guide (see the next figure). This contains a workflow for acquiring data, especially suited for new or occasional users. In Automatic mode, the Acquisition Guide will simply execute an acquisition command when you click the corresponding button. This requires the acquisition parameters to be set correctly. In interactive mode (Automatic mode unchecked), the Acquisition Guide will, at each step, open a dialog box offering you the available options and required parameters.
