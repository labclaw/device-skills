# parameterindex

**Category:** Commands > Miscellaneous > TopSpin Interface

## NAME

**parameterindex** - Open the parameter index


## DESCRIPTION

The command parameterindex opens a parameter index dialog box.
 
It displays all TopSpin parameters which can be entered in various menus and tables as well as in the command line. The following actions are available:
Help
Open the HTML Help page of the selected parameter. This is equivalent to double-clicking the parameter.
Execute
A parameter setting box will be opened in which the selected parameter can be set.
Find
Find a character string in the parameter index.
