# tfp, tsfp

**Category:** Commands > Process > Advanced > Special Transforms

## NAME

**tfp** - Traficante window mult. + FT + phase correction (1D)

**tsfp** - Traficante + S / N enhancement + FT + phase correction (1D)


## DESCRIPTION

The commands tfp and tsfp are similar to the command efp. They use the Traficante window multiplication instead of the exponential window multiplication followed by a Fourier transformation, and phase correction with/without S / N (signal-to-noise) enhancement.
