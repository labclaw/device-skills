# masreflip

**Category:** Commands > Acquire > Spin

## NAME

**masreflip** - Flips the status without ejecting the rotor


## DESCRIPTION

The masreflip command flips the stator without ejecting the rotor. It can be used to make sure that the stator flips reproducible after the readjustment of the magic angle position.
