# setshim, setsh

**Category:** Commands > Acquire > Shim

## NAME

**setshim** - Set Shims

**setsh** - Set Shims


## DESCRIPTION

The setshim command allows to set shim values individually from a graphical user interface that shows all available shims. The command setsh is the same as setshim.
 
The same functionality is also available through the command bsmsdisp.
