# masdisp

**Category:** Commands > Acquire > Spin

## NAME

**masdisp** - Open interface to control the MAS unit


## DESCRIPTION

The user interface can be accessed in different ways:
Enter masdisp at the TopSpin prompt.
Click mas in the TopSpin Acquisition tab.
Click the MAS spin rate icon  on the TopSpin acquisition status bar at the bottom of the TopSpin window.
