# topshim, topshim gui, topshim report

**Category:** Commands > Acquire > Shim

## NAME

**topshim** - Start Automatic Shimming


## DESCRIPTION

1. topshim command starts the automatic shimming procedure.
2. topshim gui opens the TopShim graphical user interface.
3. topshim report opens the TopShim report from the last TopShim run. All report files from previous TopShim runs are stored in the TopSpin log files directory located in <tshome>/prog/curdir/<user>.
For more information about the TopShim procedure please refer to the TopShim User Manual under Help | Manuals | Acquisition – Application Manuals | TopShim.
