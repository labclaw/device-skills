# copypars

**Category:** Commands > Acquire > Set Limits

## NAME

**copypars** - Advanced transfer of parameters


## DESCRIPTION

Provides an interface for copying sweep widths, excitation frequencies and other acquisition parameters from one dataset to another. Both a manual mode providing you the option to select the specific parameters to copy and a fully automatic mode are available.
