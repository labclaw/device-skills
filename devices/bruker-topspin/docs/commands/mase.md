# mase, maseject

**Category:** Commands > Acquire > Spin

## NAME

**mase** - Eject rotor from MAS probe

**maseject** - Eject rotor from MAS probe


## DESCRIPTION

The mase command ejects the rotor from the MAS probe into the rotor transfer system. The rotor ends up in the catcher or in the automation device, depending on which hardware is installed.
