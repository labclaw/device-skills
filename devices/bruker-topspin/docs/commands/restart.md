# restart

**Category:** Commands > Miscellaneous > TopSpin Interface

## NAME

**restart** - Restart the TopSpin Graphical Interface


## DESCRIPTION

This command restarts the TopSpin Graphical Interface (the Java virtual machine). 
This command is useful if some graphical elements are not displayed correctly, or the TopSpin Graphical Interface is suspected to not work correctly. A restart of the TopSpin Graphical Interface does not stop a running acquisition.
If the TopSpin Interface does not respond at all to commands or mouse clicks, it may be restarted with the external command restartgui.
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
