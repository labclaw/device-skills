# qnmr

**Category:** Commands > Analyze > Quantify

## NAME

**qnmr** - Potency determination


## DESCRIPTION

PotencyMR is a workflow for potency determination offering the design of methods and a straightforward and automated application of them. The potency determination is streamlined from sample deposition in the sample changer to the potency report.
Potency determination consists of the determination of the % weight/weight of an analyte in a given substance.
### NOTE

This command is only available in IconNMR and as part of CMC-assist. It cannot be started from TopSpin.
For further information please refer to the NUS User Manual under Help | Manuals | Automation and Data Publishing | PotencyMR.
