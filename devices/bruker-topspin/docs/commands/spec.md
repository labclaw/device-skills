# spec

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**spec** - Switch to Spectrum Display


## DESCRIPTION

The command spec switches to the spectrum display in the current dataset window. It is equivalent to clicking the SPECTRUM Tab in TopSpin.
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
