# adiag

**Category:** Commands > Manage > Spectrometer

## NAME

**adiag** - Setup and running of the automatic diagnosis program (remote monitoring)


## DESCRIPTION

The command adiag provides access to the setup and running of the automatic diagnosis program, also known as the remote monitoring feature of TopSpin.
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
