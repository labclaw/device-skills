# ii

**Category:** Commands > Acquire > Run

## NAME

**ii** - Initialize spectrometer interface


## DESCRIPTION

The command ii initializes the spectrometer interface. It tries to access all hardware parts of the spectrometer that are needed for the current experiment and loads the acquisition parameters.
ii must be executed once, when the spectrometer has been switched off.
For AVANCE spectrometers it is possible to effectuate a restart of the hardware controllers with the command ii restart. This may be helpful if a hardware failure has blocked the system.


## INPUT PARAMETERS

All acquisition parameters.
