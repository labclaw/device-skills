# proc2d, proc2d y

**Category:** Commands > Process > Processing Spectrum

## NAME

**proc2d** - Configure standard processing


## DESCRIPTION

The command proc2d opens a 2D processing dialog.
This dialog can be used for standard 2D processing, including exponential multiplication, Fourier transform, phase correction, referencing, baseline correction and plotting. Processing steps can be switched on or off and two parameters, line broadening and plot layout, can be set.
proc2d y 
Which will process the current dataset without opening the dialog, using the last settings.
