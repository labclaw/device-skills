# pdudisp

**Category:** Commands > Manage > Spectrometer

## NAME

**pdudisp** - Spectrometer Power On/Off


## DESCRIPTION

The pdudisp command opens a window from which the spectrometer can be switched on or off or rebooted.
 
The Status Messages window shows the progress of the shutdown or startup procedure. 
For more information about the spectrometer hardware and the PDU refer to the AQS technical manuals available on the BASH DVD.
