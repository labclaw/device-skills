# nmrq

**Category:** Commands > Analyze > Quantify

## NAME

**nmrq** - Quantitative NMR (1D)


## DESCRIPTION

The command nmrq starts the calculation based on integrals, molecular weights, and component weights. From this information, the purity of one or more components can be calculated. Click Analyse | Quantify | Quantitative Analysis Tutorial for an explanation of the method and the usage of the nmrq command.
