# mana

**Category:** Commands > Analyze > Multiplets

## NAME

**mana** - Switch to Interactive Multiplet Analysis Window (1D)


## DESCRIPTION

The command mana switches to multiple analysis mode.
 
It can be started as follows:
1. Click Analyse | Multiplets. 
or 
1. enter mana in the command line 
or 
1. open it from the Multiplet Analysis Guide (command managuide).
 
A full description of the Multiplet Analysis package can be found under: 
Help | Manuals | Analysis and Simulation | Structure Analysis Tools
