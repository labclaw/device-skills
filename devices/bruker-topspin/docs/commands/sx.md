# sx ej, sx, sxl

**Category:** Commands > Acquire > Sample

## NAME

**sx** - Change Sample according to Sample position

**sxl** - Change Sample according to Barcode Label


## DESCRIPTION

The sx commands change the sample in the magnet. 
1. sx ej ejects the sample using the functionality provided by the connected sample changer.
2. sx asks for the sample to be inserted. It will eject the currently loaded sample and replace it with the one from the position provided in the dialog window.
 
1. sxl works like sx but uses the specified barcode label to identify the sample to be inserted.
