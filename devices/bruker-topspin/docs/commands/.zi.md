# .f1r, .f2r, .hr, .zi, .zo, .zl, .zx

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**.f1r** - Reset F1 Zooming to Full Spectrum (2D, 3D)

**.f2r** - Reset F2 Zooming to Full Spectrum (1D, 2D, 3D)

**.hr** - Reset Zooming (horiz. Scaling) to Full Spectrum (1D, 2D, 3D)

**.zi** - Zoom In (1D, 2D, 3D)

**.zo** - Zoom Out (1D, 2D, 3D)

**.zl** - Retrieve Previous Zoom (1D, 2D, 3D)

**.zx** - Exact Zoom via Dialog Box (1D, 2D, 3D)


## DESCRIPTION

The commands zoom in or out of the displayed region of the dataset.
