# fid

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**fid** - Switch to FID Display


## DESCRIPTION

The command fid switches to the FID display in the current dataset window. It is equivalent to clicking the FID Tab in TopSpin.
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
