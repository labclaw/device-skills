# cmcse

**Category:** Commands > Applications > Small Molecules

## NAME

**cmcse** - Structure Elucidation


## DESCRIPTION

The program generates possible structures based on the information from analyzed 1D and 2D NMR spectra. Example datasets exam_CMCse_1 (alpha-Ionone), exam_CMCse_2 (Guaiol), and exam_CMCse_3 (Strychnine) are delivered as part of the TopSpin example datasets. CMC-se comes with its own documentation, type help cmcse-man or click Help | Manuals | Analysis and Simulation | CMC-se Structure Elucidation.
