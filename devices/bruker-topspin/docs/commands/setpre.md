# setpre

**Category:** Commands > Manage > Spectrometer > Experiments/Parameters

## NAME

**setpre** - Set all gradient related parameters


## DESCRIPTION

The setpre command is used to set all gradient related parameters. It is especially important if the spectrometer is equipped with two different types of gradients, for instance, in diffusion NMR systems.
 
For further information please refer to the Diffusion User Manual under Help | Manuals | Acquisition - Application Manuals | Diffusion.
