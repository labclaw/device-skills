# .winf, .ph, .basl, .cal, .pp, .int, mana, .md, .seto123

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**.winf** - Switch to Interactive Window Multiplication (1D)

**.ph** - Switch to Interactive Phase Correction Mode (1D, 2D)

**.basl** - Switch to Interactive Baseline Correction Mode (1D)

**.cal** - Switch to Interactive Spectrum Referencing Mode (1D, 2D)

**.pp** - Switch to Interactive Peak Picking Mode (1D, 2D, 3D)

**.int** - Switch to Interactive Integration Window (1D, 2D)

**mana** - Switch to Interactive Multiplet Analysis Window (1D)

**.md** - Switch to Multiple Display Mode (1D, 2D, 3D)

**.seto123** - Switch to Interactive SFO1 Setting Window


## DESCRIPTION

These commands switch from the main spectrum display to a dedicated task window. 
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
