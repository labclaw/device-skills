# xpy3

**Category:** Commands > Miscellaneous > Run a Program

## NAME

**xpy3** - Execute Python 3 program


## DESCRIPTION

Python 3 scripts use their own Python environment. (For comparison, often mentioned Jython is a Python 2.7 dialect. Jython scripts run together with TopSpin GUI in a shared Java environment).
The command xpy3 opens a dialog where you can select the desired Python program:
Path is a field where you can enter the full path name of the Python program. Click Execute to run it.
### Parameters, Lists, AU Programs

Python programs can also be executed from the command line by entering the script name:
e.g.:
xpy3 ExamCmd4.py 
xpy3 publish /tmp/mypicture.svg
