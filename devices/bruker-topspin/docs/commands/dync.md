# dync

**Category:** Commands > Applications > Dynamics

## NAME

**dync** - Dynamics Center


## DESCRIPTION

The command dync starts the Dynamics Center software package. The package must be installed separately from the Bruker web. It contains a general dynamics part that is free to use. The Dynamics Center also contains the former Protein Dynamics Center, which requires a PROTEIN_DYNAMIC2 license. A detailed description as well as a tutorial is available from the Bruker web.
