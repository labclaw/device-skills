# account

**Category:** Commands > Manage > Spectrometer > Save/Restore Installation

## NAME

**account** - Show acquisition accounting statistics


## DESCRIPTION

The command account prints statistics derived from the experiments performed during a desired time.
Statistics are only available if accounting is switched on. You can do this as follows:
1. Click Options | Preferences [set].
2. Click Acquisition in the left part of the dialog box.
3. Click Change on Configure accounting & data archiving after 'zg' and enable Write accounting info after 'zg' .
4. Click OK.
### INPUT FOLDERS

<tshome>/prog/curdir/acqhistory - acquisition accounting files
