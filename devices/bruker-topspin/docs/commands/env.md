# env

**Category:** General Information

## NAME

**env** - Set Environment Variable for TopSpin Commands


## DESCRIPTION

The command env allows to set (or get) environment variables for TopSpin commands. For instance, the dataset conversion commands jconv and vconv will look for datasets to be converted in the directories pointed to by the environment variables JNMR and VNMR. Typing env from the TopSpin command line shows the command usage.
