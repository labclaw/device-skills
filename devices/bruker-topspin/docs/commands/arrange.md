# arrange

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**arrange** - Arrange Data Windows as a Grid


## DESCRIPTION

If more than one data set has been opened in TopSpin, then arrange shows these data windows in a grid-like representation. It is equivalent to clicking the Show Layouts Options button and then selecting the Grid Display button.
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
