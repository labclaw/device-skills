# t1guide

**Category:** Commands > Applications > Dynamics

## NAME

**t1guide** - Open the relaxation analysis guide (2D)


## DESCRIPTION

The command t1guide opens a dialog box with a workflow for relaxation analysis including T1/T2. 
 
Although still available, the procedure is somewhat outdated as it is also available through a workflow button bar.
Click Applications | Dynamics | T1T2 (t1t2). The procedure is completely described in the TopSpin Users Guide. For more information, refer to the User Guide under:
