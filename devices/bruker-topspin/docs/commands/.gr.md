# .dist, .f1f2region, .gr, .hz, .keep, .ls, .lt, .lv, .setsw, .sync, .y, .zoommode

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**.dist** - Start Distance Measurement

**.f1f2region** - Saves the Limits of the Displayed Region to Various Parameters for Other Commands (1D, 2D)

**.gr** - Switch between Different Grid Modes (1D, 2D, 3D)

**.hz** - Toggle between Hz and ppm Axis Units (1D)

**.keep** - Keep Hor. / Vert. Scaling when Changing Datasets (1D, 2D, 3D)

**.ls** - Store Contour Levels (2D, 3D)

**.lt** - Toggle between positive and negative level display (2D, 3D)

**.lv** - Edit Contour Levels (2D, 3D)

**.setsw** - Sets the Sweep Width Parameter SW to the Currently Displayed Spectrum Region

**.sync** - Adapt Visible Region of all Data Windows to Active One

**.y** - Switch the Y-Axis Display between abs / rel / off (1D)

**.zoommode** - Change the Zoom Mode from “Dragging” to “Box” Mode
