# csearch_ident

**Category:** Commands > Applications > Small Molecules

## NAME

**csearch_ident** - Identify C13 Spectra with CSEARCH


## DESCRIPTION

This command creates an E-Mail that, once completed, will be sent to the CSEARCH library in Vienna. Based on the peak list provided in the E-Mail, a search for matching structures will be initiated and links to possible structures will be returned to the sender. The input dialog for setting up the E-Mail looks like this.
 
It is advisable to eliminate solvent peaks from the peak picking result. The screenshot below shows the 13C spectrum of Menthyl anthranilate from the Bruker example dataset Menthyl-Anthranilate/11.
 
A search on this 13C spectrum will return an E-Mail containing links to an HTML-file as well as a PDF-file where the results are summarized, and all possible matching structures are listed and ranked by probability. See here the first page of the search for Menthyl anthranalite.
