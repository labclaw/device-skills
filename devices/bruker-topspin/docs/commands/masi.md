# masi, masinsert

**Category:** Commands > Acquire > Spin

## NAME

**masi** - Insert rotor into the MAS probe

**masinsert** - Same as masi


## DESCRIPTION

The masi command inserts the rotor into the MAS probe and flips the stator into the magic angle position. The rotor must be positioned on top of the rotor transfer systems before one can use masi.
