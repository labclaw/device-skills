# nmrsim

**Category:** Commands > Applications > Simulate

## NAME

**nmrsim** - Simulate FID from Pulse Program


## DESCRIPTION

The NMRSim program is part of TopSpin and starts a new window (outside of TopSpin) from which parameters and files can be defined to simulate an FID based on the selected pulse program and spin system. The simulated FID is stored as TopSpin dataset and can then be inspected and processed inside TopSpin.
The command nmrsim comes with its own documentation, type help nmrsim-man or click Help | Manuals | Analysis and Simulation | NMRSim Experiment Simulator.
