# cascade

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**cascade** - Cascade Data Windows


## DESCRIPTION

If more than one data set has been opened in TopSpin, then cascade shows these data windows in a cascaded representation.
 
For further information please refer to the User Manual under Help | Manuals | General | User Manual.
