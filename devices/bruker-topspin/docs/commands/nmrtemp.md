# nmrtemp

**Category:** Commands > Acquire > More

## NAME

**nmrtemp** - NMR Thermometer


## DESCRIPTION

The command nmrtemp opens a workflow button bar for the setup and running of temperature-controlled acquisitions.
 
A detailed description of nmrtemp can be found under Help | Manuals | Acquisition | Application Manuals | NMR Thermometer.
