# cf

**Category:** Commands > Manage > Spectrometer > Hardware Detections

## NAME

**cf** - Configure the spectrometer


## DESCRIPTION

The command cf configures your spectrometer. It must be executed after a new version of TopSpin has been installed or if your spectrometer hardware has changed.
