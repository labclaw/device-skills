# .sl, .sl0, .sr, .sr0, .sd, .su

**Category:** Commands > Miscellaneous > Graphical User Interface

## NAME

**.sl** - Shift to Left by Half of the Displayed Region (1D, 2D, 3D)

**.sl0** - Shift to Extreme Left, Showing First Data Point on the Left (1D, 2D, 3D)

**.sr** - Shift Right by Half of the Displayed Region (1D, 2D, 3D)

**.sr0** - Shift to Extreme Right, Showing First Data Point on the Right(1D, 2D, 3D)

**.sd** - Shift the Spectrum Vertically Down by Half of the Displayed F1 Region (1D, 2D, 3D)

**.su** - Shift the Spectrum Vertically Up by Half of the Displayed F1 Region (1D, 2D, 3D)


## DESCRIPTION

These commands shift the spectrum to the left or right or up or down. For 1D spectra, the vertical shift up or down shifts the spectrum baseline to the middle or the bottom of the window. For 2D and 3D spectra, the vertical shift up or down shifts the spectrum by half of the currently displayed F1 region.
