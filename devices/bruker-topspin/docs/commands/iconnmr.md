# icona, iconnmr

**Category:** Commands > Acquire > More

## NAME

**icona** - Start IconNMR automation

**iconnmr** - Start IconNMR automation


## DESCRIPTION

The icona command starts the icon driven user interface IconNMR for routine spectroscopy and automation. The command iconnmr is the same as icona.
For more information about IconNMR automation please refer to the IconNMR user manual under Help | Manuals | Automation and Data Publishing | IconNMR Automation Interface
