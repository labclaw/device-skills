# xpyp

**Category:** Commands > Miscellaneous > Run a Program

## NAME

**xpyp** - Execute Jython program defined by the parameter PYNMP


## DESCRIPTION

The command xpyp reads the parameter PYNMP from the processing parameter file of the currently active data set. It finds the program in the py subdirectory according to the search order defined in the User Preferences (command set). Then Jython is started with the so selected python program name.


## INPUT PARAMETERS

PYNMP – name of the jython program to execute.
