# t1t2

**Category:** Commands > Applications > Dynamics

## NAME

**t1t2** - Tool for the Analysis of Relaxation Type Experiments


## DESCRIPTION

The T1T2 tool is one method for the analysis of relaxation type experiments. It opens its own workflow button bar.
