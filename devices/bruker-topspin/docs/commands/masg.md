# masg, masgo

**Category:** Commands > Acquire > Spin

## NAME

**masg** - Start Spinning the MAS Rotor

**masgo** - Same as masg


## DESCRIPTION

The masg command starts the spinning of the MAS rotor until the required rotation speed defined by the parameter MASR has been reached.
For more information about the MAS unit refer to the MAS III technical manual available on the BASH DVD.
For more information about solids experiments please refer to the User Manual under Help | Manuals | Acquisition – Application Manuals | Solids.
