# setlimits

**Category:** Commands > Acquire > Set Limits

## NAME

**setlimits** - Setup limits for 2D in F2 and F1


## DESCRIPTION

Set the limits for Homo and Hetero 2D experiments as obtained from 1D spectra.
Sets the sweep width and excitation frequency for a 2D dataset. It will prompt you to open a reference1D dataset from the TopSpin browser and zoom into the region of intereset. The SW and center of the region of the 1D spectrum will be copied to the 2D spectrum. This can be used for either homonuclear or heteronucler experiments. The nucleus of the 1D dataset used determines the dimensions(s) of the 2D to which the SW and O1 is copied.
