# cmcq

**Category:** Commands > Applications > Small Molecules

## NAME

**cmcq** - Structure/Spectra Consistency Package


## DESCRIPTION

CMC-q handles data acquisition, processing, and analysis for automated batch NMR spectroscopy. It allows for a large degree of flexibility in experimental setup within a straightforward framework. CMC-q is designed to be used in conjunction with the data analysis module CMC-assist, providing a comprehensive software environment for automated batch NMR spectroscopy. It comes with its own documentation, type help cmcq or click Help | Manuals | Acquisition – Application Manuals | CMCQ.
