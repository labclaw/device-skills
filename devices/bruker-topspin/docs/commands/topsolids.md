# topsolids

**Category:** Commands > Acquire > More

## NAME

**topsolids** - Assisted Biological Solid State NMR


## DESCRIPTION

TopSolids is an interactive user interface that is especially designed to assist in the delicate setup of probe hardware and difficult experiments dedicated to solid state NMR in structural biology and material science. It is readily accessible to users from a broad diversity of background and experience. 
For further information please refer to the TopSolids User Manual under Help | Manuals | Acquisition – Application Manuals | TopSolids.
